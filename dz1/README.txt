Программа телефонный справочник. Ввод вывод данных осуществляется через консоль. Сохраняет и читает данные из\в .json
Запуск выполняется с помощью функции user_menu()

Основные команды :
1 - Открыть файл на чтение (добавляет данные в список словарей)
2 - Сохраняет данные
3 - Показать все контакты
4 - Создать новый контакт
5 - Поиск контакта
6 - Изменить существующий контакт
7 - Удалить контакт
